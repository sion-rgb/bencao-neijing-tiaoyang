# 安全審核清單

## 安全篩查

- [ ] 所有緊急警號均觸發Level 0，停止問卷及不生成任何體質或食療結果
- [ ] 懷孕、餵哺母乳、未成年人、G6PD、一型／妊娠糖尿病等Level 1條件不顯示任何中藥、草藥或代茶飲名稱
- [ ] 二型糖尿病本身只觸發明顯注意卡，不封鎖體質結果、普通食物方向或已批准配伍
- [ ] 二型糖尿病選項不增加任何中醫體質分數，也不自動等於消渴
- [ ] 降血糖藥及胰島素與二型糖尿病分開；只顯示完成相應用藥安全欄位的Formula
- [ ] 抗凝血藥、手術、長者等其餘Level 2條件仍只顯示高度保守生活與普通食物方向
- [ ] 「不確定」或漏答安全資料會失敗關閉至Level 1
- [ ] 安全答案不進入體質計分引擎

## 選材

- [ ] 選材只來自白名單，沒有文字動態生成
- [ ] 全域禁止清單已掃描，沒有名稱、別名、複方或替代方法遺漏
- [ ] 懷孕、G6PD、糖尿病、肝腎、藥物、手術、出血、過敏欄位全部完成
- [ ] 有任何安全欄位未完成時`approved`內容不可通過build
- [ ] Formula固定包含3–6味，每味有角色及原因，不從搜尋結果或PDF藥名動態拼湊
- [ ] 未同時通過`doseReviewed`、`sourceVerified`及approved時不顯示份量
- [ ] 活血化瘀類不會對抗凝血、出血風險或手術人士顯示
- [ ] 每項有形式、時間、最長期、不適合人士、停止條件及專業確認文字
- [ ] 固定顯示「這不是中醫處方」的完整提示
- [ ] 產品設定配伍沒有冒充古方、名家原方或虛構PDF頁碼

## 專業覆核

- [ ] 中醫內容由合資格中醫專業人士覆核
- [ ] 禁忌與交互作用由具相關藥物安全能力的專業人士覆核
- [ ] 保障條款由適用司法管轄區的法律專業人士覆核
- [ ] 審核日期、版本及下次覆核日期已有紀錄
