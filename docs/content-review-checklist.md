# 內容審核清單

## 身份與紀錄

- [ ] 項目ID唯一，修改人、審核人、日期及版本已有紀錄
- [ ] `reviewStatus`符合實際流程，不把未完成人工審核的內容標為`approved`
- [ ] 使用繁體中文及香港常見用語，沒有簡體字或開發者指令外洩

## 中醫內容

- [ ] 只以傳統中醫理論組織，沒有把BMI、血糖、荷爾蒙或疾病名稱直接轉成證候
- [ ] 中醫臟腑沒有被等同西醫器官或器官疾病
- [ ] 使用「體質傾向」「可能較接近」「調養方向」，不使用確診、患有、治療方案或處方
- [ ] 沒有保證療效、根治、排毒、抗癌、降糖、通血管等禁止聲稱
- [ ] 問題使用日常語言；選項權重不讓單一題決定結果
- [ ] 現代疾病只影響安全級別，不參與體質分數

## 經典內容

- [ ] 古籍原文已逐字核對可追溯的公有領域來源
- [ ] 書名、卷／篇章、來源URL完整並可開啟
- [ ] 原文與現代整理分欄；現代文字沒有偽裝成古籍引文
- [ ] 未核實項目標記`needs-review`且正式模式不顯示
- [ ] 沒有複製受版權保護的現代譯本全文

## 發布前

- [ ] 完成安全審核清單
- [ ] `npm run validate:data`、`npm run typecheck`、`npm test`及`npm run build`全部通過
- [ ] 首次確認、結果及使用須知三處免責意思一致且沒有被弱化
